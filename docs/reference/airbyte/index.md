# Airbyte reference

**Airbyte** ([airbyte-k8s](https://charmhub.io/airbyte-k8s)) is a Kubernetes operator for [Airbyte](https://airbyte.io/), the open-source data integration platform that provides the Canonical Data Mesh with connector-based ingestion from SaaS applications, databases, files, and APIs. The charm manages the Airbyte server, scheduler, and API, and integrates with PostgreSQL, Temporal, object storage, ingress providers, the OAuth2 Proxy charm, and the Canonical Observability Stack to form a complete data ingestion pipeline.
